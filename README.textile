h1. youtube-api

Google App Engine service used as the backend for "YouTube YQL tables":http://github.com/yql/yql-tables/tree/master/youtube/

h2. How to use it:

h3. Get video metadata by id

*Parameters:*
  * *id:* YouTube's video id. For instance, the id would be *gmvQ1uA202M* for the video with url=http://www.youtube.com/watch?v=gmvQ1uA202M.

Example: "http://youtube-api.appspot.com/video?id=Lq35LACdErI":http://youtube-api.appspot.com/video?id=Lq35LACdErI
